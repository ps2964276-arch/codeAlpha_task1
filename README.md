# Hangman Game (Python)

This is a simple Hangman game built using Python.
The game selects a random word and the user has
limited chances to guess the correct letters.

## Features
- Random word selection
- User guesses one letter at a time
- Limited attempts (6 chances)
- Win and lose conditions

## Technologies Used
- Python
- Random module

## How to Run
1. Install Python on your system
2. Download or clone this repository
3. Open terminal / command prompt
4. Run the file using:
   python hangman.py

- List and string handling
- User input handling
- Basic game logic
